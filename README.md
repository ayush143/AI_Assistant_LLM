An AI-powered Voice Assistant built with React.js, Node.js, MongoDB, Ollama (Llama 3.1:8B), and Puppeteer. The goal of this project is to create a natural and interactive human-AI experience using voice interaction, intelligent tool calling, memory, and a 3D animated avatar.

✨ Features

- 🎙️ Voice Recognition
- 🔊 Text-to-Speech
- 🤖 3D Animated Avatar
- 🎭 Avatar Animations
- 🧠 Session Memory
- 💾 Database Memory
- 💬 Chat History
- ⚡ Real-Time AI Responses
- 🌐 Web Search & Information Retrieval
- 🛠️ Intelligent Tool Calling
- 🌍 Web Automation
- 🖥️ Application Control
- ✍️ AI Content Generation (emails, messages, essays, documents, captions, and more)
- ⏰ Smart Reminder System

🧠 What Can It Do?

The assistant can:

- Answer questions using AI.
- Search the web for up-to-date information.
- Generate emails, messages, essays, documents, and other written content.
- Control supported desktop applications.
- Automate browser tasks using Puppeteer.
- Remember previous conversations using session and database memory.
- Store and retrieve chat history.
- Create and announce reminders.
- Interact naturally through voice input and speech output.

⚙️ Workflow

Voice Input
      ↓
Speech Recognition
      ↓
Memory Retrieval
      ↓
AI Processing (LLM)
      ↓
Tool Calling (if required)
      ↓
Web Search / Automation / Content Generation / Reminder
      ↓
Voice Response
      ↓
Avatar Animation
